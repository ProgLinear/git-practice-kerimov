Bu Readme.md faylında github və verilən git kommandalarını izzah edəcəm.

---

## git nədir?
git versiya idarə etmə sistemidir. Kodda edilən dəyişiklikləri qeyd edir.
Bu texnologiyadan *Github, Azure* kimi birçox texnologiya istifadə edir

## Github nədir
Github git texnologiyasından istifadə edən kod paylaşma platformasıdır. Burada şirkətlər, fərdi şəxslər, bir sözlə hərkəs istənilən kod lahiyəsini paylaşa bilər.

---

## istifadə olunan terminlər və kodlar
- *repository (repo)* - Lahiyənin kodlarının olduğu yer
- *branch* - *bilmirəm*
- *commit* - dəyişiklikləri yadda saxlayır və qeyd edir
- *remote* - uzaqdan əlaqə
- *push* - kodun serverə yüklənilməsi
- *clone* - kodun serverdən alınmasa 

---


## Dərsdə istifadə olunan ***git*** kommandlarə və açıqlaması

- `git init` - Yeni Git repository yaradır
- `git add README.md` - Repo-ya README.md faylını əlavə edilir.
- `git add .` - `.` istifadə olunarsa bütün fayllar əlavə edilir.
- `git commit -m "first commit"` - "first commit" adlı commit yaradılır.
- `git branch -M main` - main adlı branch yaradır.
- `git remote add origin https://github.com/ProgLinear/notimportant.git` - github hesabı ilə əlaqə yaradır.
- `git push -u origin main` - Lokal kodları gitHub-a göndərir
